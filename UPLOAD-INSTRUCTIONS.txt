BLISKO - FLAT GITHUB PAGES VERSION

This version is intentionally flat: every file belongs in the root of your GitHub repository.

1. Delete the existing files from the blisko-polish-app repository, or create a new repository.
2. Extract this ZIP.
3. Upload ALL extracted files directly to the repository root.
4. Confirm that index.html, styles.css, app.js, data.js, sw.js and the icon files are all visible on the first page of the repository.
5. In GitHub: Settings > Pages > Deploy from a branch > main > /(root) > Save.
6. Wait a few minutes and open:
   https://karzz93.github.io/blisko-polish-app/
7. Refresh once. If the old unstyled version appears, clear site data or uninstall the old PWA and reopen the URL.
8. In Chrome choose Install app.

Do not upload the ZIP itself. Upload the files extracted from it.
